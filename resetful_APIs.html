FIVE MAIN HTTP VERBS

  • GET
  • DELETE
  • POST
  • PUT / PATCH

    • Get - requesting some data(from db, html doc, etc) Post - submitting data (stored in db, manipulated via api, etc) Put/Patch - updating data already in database Put - updates entire object Patch - updates parts of an object Delete/destroy - deletes data

    RESTFUL routing

    REST = representational state transfer
